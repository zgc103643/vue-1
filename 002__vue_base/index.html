Vue.js 配置环境

  1. cnpm init 初始化项目
  2. 在example文件夹下建立项目所需要的文件
  3. cnpm install -g live-server 全局安装live-server
  4. live-server 即可打开本地 http://127.0.0.1:8080 端口,实现自动监听
